Hello! 👋 I’m Gayatri Sanap.

📌 Project Overview

Financial Jarvis Assistant is a smart financial automation system designed to help companies manage, analyze, and understand their financial data easily.

It works like a simplified version of Power BI, combined with an intelligent AI assistant that can answer questions based on your financial data.

🚀 Features
📂 Monthly Data Upload
Upload company financial data for each month using Excel/CSV files.
📊 Automatic Data Visualization
The system processes uploaded data and generates:
Charts
Graphs
Dashboards
Insights
📅 Month-wise Organization
Data is structured for all 12 months, making it easy to track financial performance over time.

🤖 AI Financial Assistant
Ask questions like:

“What is the total expense in January?”
“What are the income sources for March?”
“Compare expenses between two months”

The assistant understands your query and provides smart answers based on the uploaded data.

📈 Interactive Dashboard
A user-friendly dashboard to view and analyze financial information visually.
⚙️ How It Works
Upload monthly financial data (Excel/CSV format).
The system stores and processes the data.
Charts and visual reports are automatically generated.
Use the AI assistant to ask questions and get insights from the data.
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
AI Integration: Gemini API 
Data Handling: CSV/Excel Processing
Visualization: Charts & Dashboard UI
🎯 Purpose

This project aims to:

Simplify financial data analysis
Reduce manual effort in reporting
Provide quick insights using AI
Help businesses make better decisions
